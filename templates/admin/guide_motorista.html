Guia para Configuração de Acesso de Motoristas

Como liberar o acesso para motoristas no TransportadorPRO

Para configurar um usuário como motorista e permitir acesso ao painel de motorista, siga os passos abaixo:

  1. Crie ou edite um usuário
    • Acesse: Usuários
    • Clique em "Adicionar usuário" ou selecione um usuário existente
    • Preencha as informações básicas (nome, usuário, senha)
  2. Configure o perfil como MOTORISTA
    • Após criar ou selecionar um usuário, role para baixo até a seção "Perfil do Usuário"
    • No campo "Tipo de Usuário", selecione "Motorista"
    • Preencha os campos específicos para motoristas:
      • CNH
      • Categoria CNH
      • Validade CNH
      • Selecione um caminhão (opcional)
  3. Salve as alterações
    • Clique no botão "Salvar" no final da página
  4. Confirme que o usuário está ativo
    • Verifique se a opção "Ativo" está marcada no perfil do usuário
    • Verifique se a opção "Ativo" está marcada nas informações básicas do usuário

Observações importantes:

  • Apenas usuários com o perfil tipo "MOTORISTA" terão acesso à área de motoristas
  • O usuário deve estar ativo para poder acessar o sistema
  • O caminhão atribuído ao motorista aparecerá no dashboard do motorista
  • Um motorista pode gerenciar apenas os abastecimentos relacionados ao caminhão atribuído a ele

Links úteis:

  • Gerenciar caminhões
  • Gerenciar abastecimentos pendentes
